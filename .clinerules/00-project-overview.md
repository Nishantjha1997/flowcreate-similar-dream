## Project Name

## Description
A full-stack resume builder application with an integrated Applicant Tracking System (ATS). Users can create professional resumes using multiple templates, customize styling, and export to PDF. Organizations can use the ATS module to post jobs and manage candidates.

## Core Modules

### 1. Resume Builder
- Multiple professional resume templates (Modern, Classic, Creative, Technical, Professional)
- Real-time live preview with instant updates
- Customization panel for colors, fonts, spacing, layout
- PDF export functionality using html2pdf.js
- Profile data sync - fill resume from user profile
- Section drag-and-drop reordering

### 2. User Profile Management
- Comprehensive profile with personal, professional, experience, education, skills, projects, certifications, volunteer sections
- Profile completeness tracking with percentage indicator
- Auto-save functionality
- PDF resume upload with AI data extraction (requires Gemini API key)
- Profile to resume sync

### 3. ATS (Applicant Tracking System)
- Organization onboarding and management
- Job posting creation and management
- Candidate application pipeline
- Interview scheduling
- Talent pools
- Multi-user team collaboration with roles

### 4. Authentication & Authorization
- Supabase Auth integration
- Email/password authentication
- Role-based access (admin, moderator, user)
- Organization-based access control for ATS

### 5. Premium Features
- Razorpay payment integration
- Subscription management
- Premium templates and features
